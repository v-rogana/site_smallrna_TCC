About the Research

Turning the host immune response into a virus discovery engine

When viruses infect insects, the host RNA interference (RNAi) pathway produces small RNAs (18–30 nt) that carry a unique signature for each virus — a molecular fingerprint shaped by genome structure, tissue tropism, and replication strategy.

Our approach exploits these virus-derived small RNAs to detect and characterize viruses without relying on sequence similarity to known references. This overcomes a critical limitation of traditional metagenomics: the inability to identify highly divergent viruses with no known relatives — the so-called "dark matter" of viral metagenomics.

The method has been validated across Drosophila, Aedes aegypti mosquitoes, and Lutzomyia sandflies, and preliminary results extend to plants and vertebrates.

How it Works

From raw reads to viral discovery

The pipeline processes small RNA sequencing libraries through integrated stages — combining classical bioinformatics with machine learning to identify viruses that escape traditional detection.

RNA Preparation
Input
Total RNA extracted from insect samples. Small RNAs (18–30 nt) are size-selected on denaturing PAGE, while long RNAs undergo poly(A) enrichment and rRNA depletion.
size selection 18–30 ntno sample manipulationTrizol extraction
Pre-processing
Filtering
Quality filtering, adaptor trimming, and sequential removal of host genome, bacterial, and transposable element sequences via Bowtie mapping.
fastx-toolkitBowtiephred ≥ 20host removal
Contig Assembly
Assembly
Parallel assembly using Velvet across 20–23 nt and 24–30 nt size ranges with optimized k-mer values. Results merged via CAP3 and deduplicated with BLASTClust.

Characterization
Dual strategy
Sequence-based
Similarity Search
BLAST and Diamond against NCBI nr/nt databases. Contigs classified as viral, non-viral, or unknown. InterProScan and HMMer identify conserved domains and ORFs.
Pattern-based
sRNA Profile Classification
Small RNA size profiles are Z-score normalized and classified by a Random Forest model trained on 1,300+ sequences — identifying viral contigs with no known relatives.
Validation
Output
All contigs classified as viral, non-viral, unknown, or EVEs — validated by RT-PCR and Sanger sequencing.

Companion Tools

Processing BLAST and Diamond results for viral metagenomics

Beyond the core small RNA Metavir pipeline, this companion repository provides scripts and workflows for processing tabular BLASTn and DIAMOND results — a critical step in any viral metagenomics analysis.

It covers the full workflow from downloading and formatting NCBI databases with taxonomy information, to filtering hits, enriching FASTA headers with alignment metadata, and adding complete taxonomic lineages to results.

3
Run BLASTx with rich output
Generate tabular results with coverage, identity, e-values, strand info, and taxonomy IDs

Filter & annotate

Example: DIAMOND blastx with taxonomy 
# Build taxonomy-aware database
diamond makedb \
  --in refseq_protein.fasta \
  -d refseq_protein_tax \
  --taxonnodes nodes.dmp \
  --taxonnames names.dmp \
  --taxonmap prot.accession2taxid.FULL \
  --threads 80

# Run sensitive search with rich output
diamond blastx \
  -q contigs.fasta \
  -d nr_tax.dmnd \
  -k 10 -p 31 -e 0.001 \
  -f 6 qseqid qlen qstart qend \
     qcovhsp length sseqid slen \
     sstart send scovhsp pident \
     evalue bitscore qstrand \
     qframe stitle staxids \
  --very-sensitive \
  -o hits_out.tab \
  --al aligned.fasta \
  --un unaligned.fasta
